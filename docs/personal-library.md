# 模块规格：个人学习库

## 模块定位

个人学习库用于保存和整理用户自己的学习资料。

第一版不强制用户保存题目时写总结。物理竞赛学生不应该被过度强制进入普通学习流程。

个人学习库应该提供自由整理空间。

## 第一版能力

个人学习库支持：

- 查看已收藏题目
- 管理题目收藏
- 创建个人文章 / 文档
- 查看个人文章 / 文档
- 编辑个人文章 / 文档
- 创建题集
- 创建文件夹
- 把题目放入题集
- 把题集放入文件夹
- 把文章 / 文档放入文件夹

## 最小对象

个人学习库第一版最小支持对象：

- 题目收藏
- 个人文章 / 文档
- 题集
- 文件夹

题集和文件夹不是同一种对象。

题集类似一种文件，里面放很多题。

题集可以被放进文件夹。文章 / 文档也可以被放进文件夹。

AI 解题会话记录不直接显示在个人学习库中。AI 解题会话记录属于 AI 解题工具模块。

## 文章与个人学习库

用户创建的文章 / 文档属于个人学习库。

它可以是私有，也可以公开到文章广场。

公开后，它仍然是同一个文档。

修改个人学习库中的这个文档，会影响公开文章。

## 数据需求

个人学习库需要维护：

- 用户
- 题目收藏
- 题集
- 文件夹
- 个人文章 / 文档
- 文档公开状态
- 文档与文章广场的关系
- 题目与题集的关系
- 题集与文件夹的关系
- 文档与文件夹的关系
