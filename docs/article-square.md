# 模块规格：文章广场

## 模块定位

文章广场是第一版正式模块，用于让学生和管理员发布物理竞赛相关内容。

第一版文章广场要有基本功能，但不是完整知乎式社区。

## 第一版支持

文章广场支持：

- 查看文章列表
- 搜索文章
- 阅读文章
- 写文章
- 发布文章
- 文章关联一个或多个题目
- 文章不关联题目
- 公开文章时选择知识点 / 模型标签

第一版不做：

- 文章审核
- 评论
- 点赞
- 复杂收藏
- 推荐系统
- 复杂引用来源标注

加入个人学习库可以视为一种收藏方式。

## 文章对象

文章和个人文档是同一个对象。

文章可以是私有，也可以公开。公开后，它出现在文章广场。

公开后修改这个文档，会影响公开文章本身。

产品关系类似：

```text
一个文档可以是 private
也可以被设置为 public
公开后别人可以看到
修改它就是修改公开内容
```

## 文章与题目关系

文章不一定要关联题目。

文章可以关联一个或多个题目。

文章正文中任意位置可以嵌入题目内容，并跳转到题目详情。

题目保存在数据库中，因此文章通过题目链接跳转到题目详情。

## 文章标签

文章公开时需要选择：

- 知识点标签
- 模型标签

这些标签用于文章广场中的搜索和组织。

是否必填仍未最终确认，见 `docs/product/open-questions.md`。

## 数据需求

文章广场需要维护：

- 文章 / 文档
- 作者
- 公开状态
- 文章内容
- 知识点标签
- 模型标签
- 关联题目
- 正文中嵌入的题目链接
- 发布时间
- 修改时间
